### Installation

~~~bash

git clone https://github.com/thespartoos/AutoBspwm

cd AutoBspwm

chmod +x autoBspwm.sh

# run Autobspwm script #

./autoBspwm.sh

# select theme

./autoBspwm.sh -t forest

# change them

./autoBspwm.sh -c deep

~~~
